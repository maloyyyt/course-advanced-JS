# Instapro

Учебное MVP-приложение социальной сети для публикации и просмотра фотографий.

## Реализованный функционал

* Получение и отображение общей ленты публикаций через API.
* Создание новых постов с возможностью загрузки изображения и добавления описания.
* Просмотр публикаций выбранного пользователя с переходом из общей ленты.
* Возможность ставить и убирать лайки без перезагрузки страницы.
* Автоматический подсчёт количества лайков на основе данных API.
* Отображение времени публикации в относительном формате.
* Защита от XSS-атак за счёт экранирования пользовательского контента.
* Единая шапка сайта на всех страницах приложения.
